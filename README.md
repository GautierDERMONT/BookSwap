# BookSwap

## Description
BookSwap est une application web qui permet aux utilisateurs d’échanger des livres facilement.

## Stack technique
- **Frontend** : React  
- **Backend** : Node.js  
- **Base de données** : MySQL  
- **Authentification** : JWT (JSON Web Token)
